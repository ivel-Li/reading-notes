# 📚 读书笔记知识图谱

基于 **Quartz 4** 构建的读书笔记可视化知识库，支持双向链接、知识图谱、全文搜索。

## 项目定位

为**小说创作**服务，围绕 8 大主题整理笔记：

| # | 主题 | 涵盖内容 |
|---|------|---------|
| 🏛️ | 制度与权力 | 古代政治得失、权力结构、历史兴衰 |
| 👘 | 衣食住行 | 古代服饰制度、饮食文化、居住出行 |
| 📜 | 古代文学 | 诗经、古诗十九首等古典文学 |
| ⚔️ | 战争与谋略 | 攻城方式、军备制度、谋略制定 |
| 🚀 | 人物崛起 | 从布衣到纵横捭阖的英雄 |
| 👤 | 小人物视角 | 底层个体的沉浮与命运 |
| 🔄 | 历史到文学 | 历史素材的文学转化 |
| 🐉 | 史诗奇幻 | 冰与火之歌等类型文学研读 |

## 快速开始

```bash
# 1. 安装依赖（首次）
npm install

# 2. 本地预览
npm run dev
# → 浏览器打开 http://localhost:8080

# 3. 构建静态站点
npm run build
# → 产物输出到 public/
```

> **Windows 用户注意：** 用 CMD（命令提示符），**不要用 PowerShell**。CMD 中先 `cd d:\interest\writting` 进入项目目录再运行上述命令。

## 添加笔记

在 `content/` 下新建 `.md` 文件，参照示例 `content/example-刘邦起于亭长.md` 或模板 `content/templates/笔记模板.md`：

```yaml
---
title: 笔记标题
tags: [标签1, 标签2]
created: 2025-01-01
source: 《书名》作者
---
```

- `[[另一篇笔记标题]]` 建立双向链接，知识图谱自动捕获关联
- `tags` 字段打标签，系统自动生成索引页

## 项目结构

```

│   ├── 1-制度与权力.md
│   ├── ...
