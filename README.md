# 📚 读书笔记知识图谱

基于 **Quartz 4** 构建的读书笔记可视化知识库，支持双向链接、知识图谱、全文搜索。

## 项目定位

为**小说创作**服务，围绕 8 大主题整理笔记：

| # | 主题 | 涵盖内容 |
|---|------|---------|
| 🏛️ | 制度与权力 | 古代政治得失、权力结构、历史兴衰 |
| 👘 | 衣食住行 | 古代服饰制度、饮食文化、居住出行 |
| 📜 | 古代文学 | 诗经、古诗十九首等古典文学 |
| ⚔️ | 战争与谋略 | 攻城方式、军备制度、谋略制定 |
| 🚀 | 人物崛起 | 从布衣到纵横捭阖的英雄 |
| 👤 | 小人物视角 | 底层个体的沉浮与命运 |
| 🔄 | 历史到文学 | 历史素材的文学转化 |
| 🐉 | 史诗奇幻 | 冰与火之歌等类型文学研读 |

## 添加笔记

在 `content/` 下新建 `.md` 文件，参照示例 `content/example-刘邦起于亭长.md` 或模板 `content/templates/笔记模板.md`：

```yaml
---
title: 笔记标题
tags: [标签1, 标签2]
created: 2025-01-01
source: 《书名》作者
---
```

- `[[另一篇笔记标题]]` 建立双向链接，知识图谱自动捕获关联
- `tags` 字段打标签，系统自动生成索引页

## 知识图谱主题颜色管理

图谱节点会根据笔记 `tags` 自动显示不同颜色。

### 全部已注册标签及颜色

编辑 `quartz/components/Graph.tsx`，在 `localGraph` 和 `globalGraph` 的 `themeColors` 中配置。目前注册了以下标签：

```ts
themeColors: {
  "MOC": "#6B7280",    // 目录页 - 灰色
  "制度": "#4A7FB5",   // 古代政治 - 蓝色
  "权力": "#4A7FB5",
  "服饰": "#E8833A",   // 古代服饰 - 橙色
  "衣食住行": "#E8833A",
  "地理": "#6B8E23",   // 地理宗教 - 橄榄绿
  "宗教": "#6B8E23",
  "文化": "#6B8E23",
  "人物": "#8B5CF6",   // 英雄成长 - 紫色
  "崛起": "#8B5CF6",
  "小人物": "#2D9B8E", // 小人物 - 青色
  "沉浮": "#2D9B8E",
  "战争": "#C23B3B",   // 战争谋略 - 红色
  "谋略": "#C23B3B",
  "文学": "#D97706",   // 文学 - 琥珀黄
  "史诗": "#DB2777",   // 奇幻史诗 - 粉色
  "奇幻": "#DB2777",
}
```

### 添加新主题

编辑 `quartz/components/Graph.tsx`，在 `localGraph` 和 `globalGraph` 的 `themeColors` 里各加一条：

```ts
themeColors: {
  // ...现有配置不变
  "新标签词": "#00AA00",   // ← 在这里加
},
```

### 给笔记打标签

在笔记 frontmatter 里写上对应 tag 即可：

```yaml
---
title: 唐朝官制
tags:
  - 制度
---
```

图谱中该节点会自动匹配颜色。笔记可以有多个 tag，只有以上注册过的标签才会显示指定颜色，未注册的标签显示默认灰色。

### 规则说明

1. **打标签必须用已注册的关键词**，颜色才能正确显示。
2. 一个笔记可以有多个 tag，匹配到的第一个生效。
3. 标签名 = 图谱节点名，`MOC` 标签显示为 `#MOC`（灰色），表示目录聚合页。
4. **不得在 Markdown 正文中使用 `#标签名`**（hashtag 语法），否则会生成不受颜色管理的冗余图谱节点，显示为默认色。标签统一写在 frontmatter `tags` 字段中。

| 主题 | 标签关键词 | 颜色 |

## 图谱节点名称控制

图谱里每个节点的名称由以下规则决定：

### 普通笔记节点

节点名称 = 笔记的 `title` 字段，例如：

```yaml
---
title: 刘邦起于亭长    # ← 这就是图谱里显示的名字
tags:
  - 人物
---
```

如果 `title` 没写或为空，则使用**文件名**（不含 `.md`）。

### 标签节点

标签节点在图中以 `#标签名` 显示，例如笔记里写了：

```yaml
tags:
  - 制度
  - 权力
```

图谱中会自动生成 `#制度` 和 `#权力` 两个标签节点，与相关笔记相连。

### 一句话总结

| 节点类型 | 名字来源 | 示例 |
|---------|---------|------|
| 笔记节点 | frontmatter 的 `title` | `刘邦起于亭长` |
| 标签节点 | 自动生成，前缀 `#` | `#人物`、`#制度` |
